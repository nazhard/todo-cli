# To Do CLI

1. clone repo
  - use git clone: `git clone https://github.com/nazhard/todo-cli.git`
  - use github cli: `gh repo clone nazhard/todo-cli`
  - use degit: `degit nazhard/todo-cli`
2. enter project: `cd todo-cli`
3. install: `go install`
4. build: `go build`
5. enjoy:
  ```bash
  ./todo-cli -h
  # try
  ./todo-cli -a "drink"
  ```
